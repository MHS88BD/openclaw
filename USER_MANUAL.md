# 🤖 OpenClaw AI Bot - User Manual (বাংলায়)

এই গাইডটি আপনাকে বটের সব কমান্ড এবং ফিচার ব্যবহার করতে সাহায্য করবে।

## 📱 হোয়াটসঅ্যাপ কমান্ডসমূহ (WhatsApp Commands)

বটটি হোয়াটসঅ্যাপে **Owner** (আপনার নম্বর) এবং **AI** প্রিফিক্সযুক্ত মেসেজে রেসপন্স করে।

### ১. মেম্বার এক্সট্রাকশন (Member Extraction)
- **সব গ্রুপের লিস্ট দেখতে:** 
  `ai groups`
- **গ্রুপের ভেতরে থাকা অবস্থায় সরাসরি মেম্বার লিস্ট নিতে:** 
  `ai members of this group`
- **গ্রুপ আইডি দিয়ে প্রাইভেসির সাথে (নিজের চ্যাটে) লিস্ট নিতে:** 
  `ai members <GROUP_ID>`
  *(উদাহরণ: `ai members 120363041234567@g.us`)*

### ২. স্মার্ট ব্রডকাস্টিং (Smart Broadcasting - Safe Bulk Messaging)
আপনার ব্যান হওয়ার ঝুঁকি কমাতে এই কমান্ডটি ব্যাচ আকারে মেসেজ পাঠায়।
- **কমান্ড:** 
  `ai broadcast <GROUP_ID> "<MESSAGE>" --s <START> --c <COUNT>`
- **প্লেসহোল্ডার:** আপনি মেসেজের ভেতরে `{name}` ব্যবহার করতে পারেন।
- **উদাহরণ:** 
  `ai broadcast 120363041234567@g.us "Hi {name}, কেমন আছেন?" --s 0 --c 10`
  *(এটি প্রথম ১০ জনকে ৫-১৫ সেকেন্ড বিরতি দিয়ে মেসেজ পাঠাবে)*

### ৩. ব্যক্তিগত মেসেজ পাঠানো (Send Direct Message)
ফোনে নাম্বার সেভ না থাকলেও আইডি দিয়ে মেসেজ পাঠানো যায়।
- **কমান্ড:** `ai send <ID_OR_PHONE> <MESSAGE>`
  *(উদাহরণ: `ai send 215555902590987 Hello!`)*

### ৪. আইডি ডিটেইলস চেক (ID Lookup)
কোনো আইডি বা নম্বর হোয়াটসঅ্যাপে আছে কিনা তা চেক করতে।
- **কমান্ড:** `ai lookup <ID_OR_PHONE>`

## 🚀 ব্যবহারের কিছু টিপস:
- **ব্যান এড়াতে:** ব্রডকাস্টিং করার সময় একসাথে ১০-২০ জনের বেশি পাঠাবেন না। পরবর্তী ব্যাচ পাঠানোর আগে কমপক্ষে ১ ঘণ্টা বিরতি দিন।
- **ID বনাম Phone:** কিছু মেম্বারের প্রাইভেসির কারণে ফোন নম্বর হাইড থাকে, তাদের ক্ষেত্রে `ID: xxxxxxx` দেখালে সেটিই `ai send` কমান্ডে ব্যবহার করুন।
- **Excel Formatting:** মেম্বার লিস্ট থেকে সরাসরি নম্বরগুলো কপি করে এক্সেলে পেস্ট করতে পারবেন, কারণ এগুলো `+CountryCode` ফরম্যাটে থাকে।
